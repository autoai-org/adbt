## Installation

Navigate to your themes folder in your Hugo site and use the following commands:

```
$ mkdir themes
$ cd themes
$ git clone git@github.com:themefisher/vex-hugo.git
```

## Main features

* Fully Responsive Ready.
* Powered by bootstrap 4 framework.
* Product Showcase
