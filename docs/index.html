Automatic Database Backup Toolkit

Regularly Back up databases in a simple way.

Goals

  • Support multiple databases
  • Good code quality
  • Visualized interface
  • Open Source
  • Beautiful (as much as I can achieve)

Install

Download the Binary (GUI included) from Github Releases

For CLI-Only binary, you can Get it from the Snap Store
